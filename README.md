# Adventures Kashmir — Next.js Website

Luxury Himalayan Treks & Gulmarg Ski School website built with Next.js 14, TypeScript, and Tailwind CSS.

## 🚀 Deploy to GitHub Pages

This repo is pre-configured to auto-deploy to GitHub Pages via GitHub Actions.

### One-time setup:

1. Push this repo to GitHub
2. Go to your repo → **Settings** → **Pages**
3. Under **Source**, select **GitHub Actions**
4. Push any change to the `main` branch — the site will build and go live automatically

Your site will be live at: `https://<your-username>.github.io/<repo-name>/`

## 💻 Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 🛠 Tech Stack

- [Next.js 14](https://nextjs.org/) — React framework
- [TypeScript](https://www.typescriptlang.org/) — Type safety
- [Tailwind CSS](https://tailwindcss.com/) — Styling
- [Playfair Display + Inter](https://fonts.google.com/) — Fonts
